# Тестовое задание

Верстка мини-карточки товара с адаптивом и интерактивными элементами.

## Стек технологий

- **HTML5** - верстка
- **CSS3** - стили
- **Нативный JS**
- **Swiper** - для слайдера
- **Fancybox** - модальное окно для изображений
- **БЭМ и PixelPerfect**

## Фичи (помимо ТЗ)
- **hover-состояния для кнопок** - на мой вкус
- **небольшие анимации** - кнопка "заказать" и кнопка "лайк"
- **счетчик лайков**
- **увеличенная кнопка "заказать"** - на мобильной версии
